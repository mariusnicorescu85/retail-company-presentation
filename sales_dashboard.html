üìê Business Intelligence Metrics

ROI Calculation

ROI = ((Total Sales - Total Salary) / Total Salary) √ó 100

Measures return on employee investment. Higher ROI = Better profitability

Profit Margin

Profit Margin = ((Total Sales - Total Salary) / Total Sales) √ó 100

Percentage of sales that is profit. Key indicator of business health

Efficiency Ratio

Efficiency Ratio = Total Sales / Total Salary

Sales generated per dollar of salary. Higher ratio = More efficient operations

üöÄ Getting Started

Prerequisites

  • Node.js 18+
  • Airtable account with API access
  • Airtable base with sales data

Installation

npm install

Configuration

Set environment variables:

AIRTABLE_API_KEY=your_api_key
NEXT_PUBLIC_AIRTABLE_BASE_ID=your_base_id
NEXT_PUBLIC_AIRTABLE_TABLE_NAME=Employees

Running the Application

npm run dev # Development
npm run build # Production Build
npm start # Start Production Server
